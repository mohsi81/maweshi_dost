
# 🐄 Maweshi Dost AI

AI-powered livestock assistant for farmers.

## Features
- 📷 Image analysis
- 🎤 Voice input (Urdu)
- 🧠 AI disease detection
- 🌍 Multi-language (Urdu, English, Roman Urdu)

## Tech Stack
- Gemini API (image understanding)
- Groq API (reasoning)
- Whisper (speech-to-text)
- gTTS (text-to-speech)
- Gradio UI

## Deployment
Deployed on Hugging Face Spaces
